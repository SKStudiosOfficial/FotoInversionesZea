# Sitio "Todo Computación"

Archivos:
- `index.html`
- `styles.css`
- `script.js`
- `assets/logo.svg`, `assets/favicon.svg`

## Cómo usar
1. Abre `index.html` en tu navegador.
2. Edita `script.js` y reemplaza `whatsappNumber` con tu número real (incluye el código de país, ej: `+56912345678`).
3. Cambia textos, horarios y dirección en `index.html`.
4. Reemplaza `assets/logo.svg` por tu logo real si lo deseas.

## Publicar gratis
- **GitHub Pages**: crea un repo, sube estos archivos a la rama `main` y habilita Pages desde Settings → Pages → Deploy from branch.
